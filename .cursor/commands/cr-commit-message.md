## Commit Message Rules

Create commit messages following the examples below:
Write in Japanese
Write in a way that junior engineers can understand
Output in a copy-and-paste-friendly format
Create commit messages only for staged code
For [変更内容の簡潔なタイトル], output the abbreviated prefix and, for monorepos, the package name in parentheses

### Examples

**Simple change:**

```
[変更内容の簡潔なタイトル]

- [具体的な変更点1]
- [具体的な変更点2]
- [技術的な説明や仕組みの説明（必要に応じて）]
```
